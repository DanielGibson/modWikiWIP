Quality settings

From modwiki

Jump to: navigation, search

These are the CVars and other game features that change depending on which quality level you select in the system options menu.

[edit]

Doom 3

The information in this section is specific to Doom 3.
Doom 3 Low Medium High Ultra
com_machineSpec 0 1 2 3
image_anisotropy 0 1 8 8
image_downSize 1 0 0 0
image_downSizeBump 1 0 0 0
image_downSizeSpecular 1 0 0 0
image_ignoreHighQuality 1 1 0 0
image_useCompression 1 1 1 0
image_useNormalCompression * 2 2 0 0
image_usePrecompressedTextures 1 1 1 0
r_mode 3 3 4 5
s_maxSoundsPerShader 1 1 0 0
Texture Budget 64mb 128mb 256mb 512mb


[edit]

Quake 4

The information in this section is specific to Quake 4.
Quake 4 Low Medium High Ultra
com_machineSpec 0 1 2 3
g_brassTime 0 0 1 2
image_anisotropy 1 1 4 8
image_downSize 1 0 0 0
image_downSizeBump 1 0 0 0
image_downSizeSpecular 1 0 0 0
image_ignoreHighQuality 1 1 0 0
image_useCompression 1 1 1 0
image_useNormalCompression * 2 2 0 0
image_usePrecompressedTextures 1 1 1 0
r_mode 3 3 4 5
r_skipNewAmbient 1 1 0 0
s_maxSoundsPerShader 1 1 0 0
Texture Budget 64mb 128mb 256mb 512mb

* nVidia GeForce 3 and 4MX cards use palettized normal maps in Medium and Low quality settings so this CVar is set to "1" on these cards. All other supported cards use DXT5 compression.

You can use the listImages console command to keep tabs on the texture memory usage of each setting.
