Description

This command compiles a level so it can be played and automatically loads it.

Usage

At the console type... 

testmap [Filename]

Parameters

  • [Filename] - Name of . MAP file (without the extension). If your map is in a subfolder of the / maps directory you will need to include the relative path to it, for instance: 
testmap <subfolder>/<mapname>

Notes

See the dmap console command for more information on map compilation. Testmap automatically tries to load . AAS files so you may see the error "WARNING: Couldn't load AAS file".
