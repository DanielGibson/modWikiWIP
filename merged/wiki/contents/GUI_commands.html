Commands are what actually make the GUI script come alive; they're pretty limited and execute simple tasks. They are used inside event blocks and can work with item variables as well as some item properties .

Available commands

  • endGame
  • evalRegs
  • if...else
  • localSound
  • namedEvent
  • resetCinematics
  • resetTime
  • runScript
  • set
  • setFocus
  • showCursor
  • transition
