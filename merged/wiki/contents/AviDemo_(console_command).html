Description

This command will play a . DEMO and output the frames as a . TGA images.

Usage

At the console type... 

aviDemo [Filename]

Parameters

  • [Filename] - Name of .DEMO file (without the extension).

Notes

This command will generate a lot of . TGA files, be sure to free up disk space before running it. Related CVars:

  • com_aviDemoHeight - You can select the height of each frame.
  • com_aviDemoSamples - You can select the antialiasing factor of each frame.
  • com_aviDemoTics - You can select how many frames per second should be generated.
  • com_aviDemoWidth - You can select the width of each frame.
