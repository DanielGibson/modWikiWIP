Game class

This event operates on an instance of the game class type {{{gameclass}}}

Description

Send named event string to specified gui.

Usage

void sendNamedEvent("float" "guiEnum", "string" "namedEvent" )

Parameters

  • [guiEnum] - Valid guiEnums are: GUI_PLAYERHUD, GUI_CINEMATICHUD, GUI_VEHICLEHUD
  • [namedEvent] - Event name sent to gui.

Examples

none

Notes

This script event does not return any values.
