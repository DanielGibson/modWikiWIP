An editDef from Doom3
Just like a windowDef , but accepts a text input by the user. An editable text field. Can't be used as in-game GUI.

Additional properties

  • cursorvar
  • cvar
  • cvarGroup
  • forceScroll
  • liveUpdate
  • maxchars
  • numeric
  • password
  • readonly
  • source
  • wrap

Example(s) 

editDef OG2Primary {
	rect		110, 1, 160, 14
	cvar		"ui_name"
	maxchars 	40 
	font		"fonts/bank"
	textscale	0.25
	textaligny	-6
	forecolor	0.6, 0.8, 0.8, 1
}
