A bindDef from Doom3
Used to create bindings to keys -- think of it as an editDef that only takes one key as input and that automatically binds that key to some command; the GUI system itself performs the input detection and command assigment handling.

This is, of course, used on GUIs that allow the user to change its keyboard bindings, like the control config menus on Doom 3.

Additional properties

  • bind

Example(s) 

bindDef CM1Primary {
	rect		121, 1, 195, 14
	bind		"_forward"
	font		"fonts/bank"
	textscale	0.25
	textaligny	-6
	forecolor	0.6, 0.8, 0.8, 1
}
