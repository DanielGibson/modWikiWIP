The information on this page is specific to Quake 4 .

Description

Controls which image to blend a visportals to when it's closing.

Usage

In the global section of your material shader type... 

portalImage [map]

Parameters

  • [map] - the relative path and file name to the specified portal image.

Notes

Usually the image used is "_black". See also the portaldistancefar and portaldistancenear keywords.
