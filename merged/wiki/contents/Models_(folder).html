The models directory contains numerous sub-directory. Each of these sub-directories contains files for all the models used in the game. This includes meshes, collision files and skin textures. The following file formats can be found in these folders:

Model Formats

. ASE
3DS Max ascii text based format
. LWO
native file format for Newtek's Lightwave
. MD5
proprietary format for Doom 3

Image Formats

. TGA
Uncompressed 24 or 32 bit Image Format
. JPG
Compressed Image Format
. DDS
Compressed Image Format

Other Formats

. CM
Doom 3 engine Collision Model
