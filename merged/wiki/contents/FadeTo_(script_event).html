Game class

This event operates on an instance of the game class type {{{gameclass}}}

Description

Fades to the given color up to the given alpha over the given time in seconds.

Usage

void fadeTo( vector color , float alpha , float time )

Parameters

  • [color] - The color to fade to.
  • [alpha] - The alpha value to fade to.
  • [time] - The duration of the fade.

Examples 

Â 
sys.fadeTo('1 1 1', 1, 10);

Notes

This script event does not return any values.
