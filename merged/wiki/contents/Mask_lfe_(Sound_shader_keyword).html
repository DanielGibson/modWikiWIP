The information on this page is specific to Doom 3 .

Description

The mask_lfe keywork only allows sound from the shader to be outbut through the LFE (subwoofer) surround sound speaker in a 5.1 setup.

Usage 

mask_lfe

Example 

ship_engine_rumble
{
	mindistance 100
	maxdistance 1000
	volume 10
	looping
	mask_backleft
	shakes .1
	sound/last_stand/loop_rumble_LFE_b.wav
}

Notes

When this is in use, sound only comes from that channel when 5.1 surround is used. If you use 2 channel (stereo) sound the channel will be muted and this sound will not be played.
