The information on this page is specific to Wolfenstein .

This is the folder structure of Wolfenstein. It should be noted that some of these folders are not present in the base folder itself. Instead, those folders and the assets contained therein are stored in PK4 archives .

One should also note that since the single player and multiplayer components are maintained separately, each uses it's own unique folder structure, executable, library, and assets. They are for all intensive purposes two independent programs sharing only the title Wolfenstein.

Folder structure:

Single Player

  • / base
    • / maps
    • / videos
    • / streampacks

Multiplayer

  • / base
    • / ui
