Description

This command will list all the files of the specified type found and/or loaded from the specified directory and any subdirectories. Includes files and folders within PK4 files.

Usage

At the console type... 

dirtree [folder] [extension]

Parameters

  • [folder] - The relative path from the base folder. Base is not included.
  • [extension] - Any specific file extension to display in the results.

Notes

None.
