Description

The highquality or uncompressed stage keywords cause the game to render the associated stage without using texture compression in Medium or High quality modes. Without this keyword, the map would be compressed according to the defaults for the selected quality mode. (Medium quality mode compresses bumpmaps and diffusemaps, where High quality mode only compresses bumpmaps.)

Usage 

highquality

uncompressed

Parameters

None.

Example

Forces the diffusemap stage to high quality: 

{
  blend diffusemap
  map textures/path/to/image.tga
  highquality
}

Notes

This keyword is useful for sky textures which can look very ugly when compressed.

This keyword has no effect in Low quality mode, and the forceHighQuality keyword must be used instead.

If the image_ignoreHighQuality CVar is set to 1, this keyword will be ignored.
