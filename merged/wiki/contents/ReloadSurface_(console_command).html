Description

This command reloads both the material declaration and corresponding texture images for the surface currently in the crosshairs.

Usage

At the console type... 

reloadSurface

Parameters

This command does not accept parameters.

Notes

This command is useful when testing a material shader on a surface in game.

You can make small changes to a material in general, point your crosshairs at the surface in game, and use this console command to see the effect of those changes.
