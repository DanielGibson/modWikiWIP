NOTE: Megatexture functionality is broken in the stock versions of Doom 3, Quake 4, and Prey due to a missing fragment program, megatexture.vfp. However, with a suitable replacement, the information documented in this article is still applicable. At present, you can find a mod that provides you with the necessary file here .

Description

This command generates a MegaTexture.

Usage

At the console type... 

makemegatexture [Filename]

Parameters

[Filename] - Name of . TGA file (without the extension).

Notes

This command will take a .TGA file present in the \megatexture folder and generate a new texture with four components named <filename>_preview.tga and a <filename>. MEGA file. This console command seems to be not entirely functional in Doom 3.
