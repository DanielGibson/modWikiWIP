Description

The twoSided global keyword allows a surface to be viewed from both sides.

Usage

In the global section of your material shader type... 

twoSided

Parameters

None.

Notes

By default, polygons will not be rendered at all from the reverse side, which facilitates an important optimisation known as backface culling. However, this is incorrect behaviour in some situations such as when a patch is used to represent a grating or spiderweb. In this case, the twoSided keyword is used to make the material appear from both sides.

Using this keyword also implies the noShadows keyword.
