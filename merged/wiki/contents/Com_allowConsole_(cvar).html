Description

This CVar controls how the console can be accessed.

Usage

At the console type... 

com_allowConsole [boolean]

Parameters

  • [boolean]
    • 0 - You need to press Ctrl + Alt + ~ to access the console.
    • 1 - You need to press ~ to access the console.
Note: Wolfenstein only:
(SP) Console access will pre prevented if set to 0. If set to 1, press Right Alt + Left Alt + ~ to access the console.
